# Codex Reports

This directory stores implementation reports written by Codex after completing one task from `docs/TASK_QUEUE.md`.

Reports must be written in Russian.

---

# Filename Format

Use:

```text
YYYYMMDD_HHMMSS_task_name.md
```

Example:

```text
20260702_141500_minimal_deterministic_image_director.md
```

---

# Required Report Format

Each report must include:

1. Task name
2. Date/time
3. Files created
4. Files modified
5. What was implemented
6. How it works
7. Checks/tests run
8. Weak points
9. Recommended next step
10. Whether human review is required

---

# Report Rules

- Be specific.
- Mention smoke tests and their result.
- Mention any files intentionally not changed.
- Mention temporary artifacts cleanup.
- Do not hide failed checks.
- Do not use the report to request unrelated work.
