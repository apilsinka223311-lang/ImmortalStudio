\# ImmortalStudio Architecture



Version: 1.0



\---



\# Philosophy



ImmortalStudio is built as a digital animation company.



Every AI agent has one responsibility.



No agent performs work outside of its department.



The architecture must remain modular so that any AI model can be replaced without changing the entire system.

\# Core Rule



Agents never communicate randomly.



Every request always goes through the Planner.



Planner coordinates the entire pipeline.



\---



\# Long-term Goal



The architecture must support:



\- multiple universes

\- multiple YouTube channels

\- multiple languages

\- multiple AI providers

\- local execution

\- cloud execution



without changing the overall design.
