# Visual Style

## Core Style

Cinematic dark xianxia fantasy for Shorts.

The style should feel serious, dramatic, sharp, and atmospheric.

## Overall Visual Direction

- semi-realistic anime fantasy;
- cinematic lighting;
- dark fantasy atmosphere;
- elegant sword clan aesthetics;
- ancient cultivation world;
- strong silhouettes;
- readable action for short-form video.

## Color Palette

Primary colors:

- dark blue;
- black;
- cold grey;
- silver;
- faded gold;
- muted red accents.

Azure Sword Clan visual identity:

- azure sword aura;
- pale blue robes;
- silver embroidery;
- cold stone courtyards;
- polished but severe clan architecture.

The Origin System visual identity:

- clean floating interface;
- cold white or pale blue text;
- minimal effects;
- no cute mascot;
- no game-shop interface;
- no colorful mobile game UI.

## Fallen Star Realm Visual Identity

Fallen Star Realm should feel ancient, isolated, and slightly broken.

Use:

- ruined stones;
- high mountain sects;
- misty valleys;
- broken star motifs;
- old sword marks;
- spiritual pressure;
- ancient gates;
- sealed ruins.

## Combat Visuals

Combat should be clear and sharp.

Use:

- close-ups for weak-point analysis;
- slow motion for counter windows;
- visible meridian highlights only when system analysis is active;
- sword aura flashes;
- dust, blood, broken stone, and pressure waves.

Avoid chaotic unreadable action.

## Lin Mo Visual Direction

Lin Mo should look:

- injured at the start;
- pale or exhausted;
- sharp-eyed;
- cold under pressure;
- physically weak but mentally dangerous;
- gradually more stable as recovery progresses.

Early Lin Mo should not look like an overpowered immortal lord.

## Lin Jian Visual Direction

Lin Jian should look:

- clean;
- confident;
- noble;
- disciplined;
- well-dressed;
- visibly higher status than Lin Mo.

He should feel like a dangerous young master, not a random thug.

## Forbidden Visuals

Avoid:

- childish cartoon style;
- goofy comedy faces;
- modern clothing in cultivation scenes;
- guns;
- sci-fi technology;
- colorful mobile-game shop UI;
- cute system mascot;
- generic bright superhero magic;
- oversaturated fantasy colors;
- instant overpowered aura for Lin Mo.
